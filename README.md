# April Fool Device Hack Prank 😈

A realistic mobile-friendly prank that simulates a device compatibility/security scan and then shows a fake "YOU ARE HACKED" screen before revealing **April Fool**.

## Features
- Realistic terminal-style scanning animation
- Camera permission & local selfie capture (no upload)
- Fake hack screen with countdown
- Device vibration effects
- Flash camera capture effect
- Fully responsive mobile UI
- Confetti April Fool reveal
- Share button
- Instagram follow button
- Works completely offline (HTML + CSS + JS)

## Privacy
This prank does NOT collect, store, or upload any data.  
All processing happens locally in the browser.

## How to Use
1. Open the page
2. Click **Start Test**
3. Allow camera permission
4. Watch the prank unfold 😄
